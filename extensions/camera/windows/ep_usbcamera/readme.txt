Windows USB media framework USB camera interface
notes 29.4.2020/pekka

This Windows USB camera interface code is written by Evgeny Pereguda and lightly reorganized here. 
Assember optimizations embedded in original implementation are disabled to allow 64 bit build.
The code is published with permissive CPOL license, which allows use as part of commercial license.
If this code is used, the name of author, Evgeny Pereguda, should be mentioned and copy of CPOL license
included in documentation/files.
